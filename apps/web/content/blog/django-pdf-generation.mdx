Django developers have suffered with wkhtmltopdf and ReportLab for too long. Deckle's Python SDK makes PDF generation as simple as calling an API.

## Install

```bash
pip install deckle
```

## Add to Your Django View

```python
from django.http import JsonResponse
from deckle import Deckle

client = Deckle(settings.DECKLE_API_KEY)

def generate_invoice(request, order_id):
    order = Order.objects.get(id=order_id)

    pdf = client.generate(
        html=render_to_string('invoices/template.html', {'order': order}),
        options={"format": "A4", "margin": "20mm"},
    )

    return JsonResponse({"url": pdf.url})
```

## Using Django Templates

The best part: you can use your existing Django templates. Render them to HTML with `render_to_string`, then pass the HTML to Deckle. Your existing template tags, filters, and includes all work.

## Using Deckle Templates

For production, store templates in Deckle and pass data directly:

```python
pdf = client.from_template(
    template_id="tmpl_django_invoice",
    data={
        "order_number": order.number,
        "customer": order.customer.name,
        "items": list(order.items.values("name", "quantity", "price")),
        "total": str(order.total),
    },
)
```
