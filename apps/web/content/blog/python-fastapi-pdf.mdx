Python developers deserve great PDF generation too. The Deckle Python SDK uses httpx under the hood and supports both sync and async workflows.

## Install

```bash
pip install deckle
```

## Quick Start with FastAPI

```python
from fastapi import FastAPI
from deckle import Deckle

app = FastAPI()
client = Deckle("dk_live_...")

@app.post("/generate-pdf")
async def generate_pdf(html: str):
    pdf = client.generate(html=html, options={"format": "A4"})
    return {"url": pdf.url, "pages": pdf.pages}
```
