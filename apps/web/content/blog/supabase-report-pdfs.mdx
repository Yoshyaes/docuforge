Supabase gives you a Postgres database with a great API. Deckle gives you pixel-perfect PDFs. Together, they're a powerful combo for generating data-driven reports.

## The Architecture

1. Query your Supabase database for report data
2. Pass the data to a Deckle template
3. Get back a PDF URL, stored and ready to share

## Setup

```bash
npm install deckle @supabase/supabase-js
```

## Querying Data and Generating PDFs

```typescript
import Deckle from 'deckle';
import { createClient } from '@supabase/supabase-js';

const supabase = createClient(process.env.SUPABASE_URL!, process.env.SUPABASE_KEY!);
const deckle = new Deckle(process.env.DECKLE_API_KEY!);

async function generateMonthlyReport(userId: string) {
  const { data: orders } = await supabase
    .from('orders')
    .select('*')
    .eq('user_id', userId)
    .gte('created_at', startOfMonth());

  const pdf = await deckle.fromTemplate({
    templateId: 'tmpl_monthly_report',
    data: {
      userName: orders?.[0]?.user_name,
      totalOrders: orders?.length,
      totalRevenue: orders?.reduce((sum, o) => sum + o.amount, 0),
      orders: orders,
    },
  });

  return pdf.url;
}
```

## Scheduling Reports

Use Supabase Edge Functions or a cron job to generate reports on a schedule. Deckle's batch API handles high-volume generation efficiently.
